planloft — hello

This page was published by planloft — a Claude Code / Codex plugin that hoists agent-written plans and docs into a themed, shareable store.

What just happened

  • A plan was captured to a per-project global store (~/.planloft/docs/<project>/).
  • planloft deploy rendered it to self-contained HTML with the minimal theme.
  • It was pushed to a public GitHub Pages repo and served at an unguessable URL.
  • It auto-expires after 30 days unless redeployed.

Steps it ran

  • resolve target path + theme
  • render markdown → HTML (marked)
  • ensure repo + Pages + prune Action
  • commit, push, return the link
